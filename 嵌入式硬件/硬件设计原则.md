# 硬件设计原则

## 滤波

常见的滤波形式包括：
- **单独并联电容**：旁路 / 去耦 / 高频滤波
- **RC**：低通 / 阻尼 / 吸收 / 延时
- **LC / 磁珠+C**：电源线上更强的滤波
- **RLC**：更完整但也更复杂

### 1. 仅对地并联一个 C，为何也能滤波

原因在于电容具有如下特性：
$$
X_C = \frac{1}{2\pi f C}
$$
即：
- 频率越高
- 电容阻抗越小

因此，当线路上存在高频噪声时，并联电容会为高频分量提供一条更低阻抗的路径，从而将其**旁路到地**。这就是最基础的**旁路（bypass）**或**去耦（decoupling）**。

### 2. 为什么还需要 RC

仅并联一个 `C` 虽然可以实现滤波，但效果取决于其前级“源阻抗”的大小。
这一点非常重要。

#### 2.1 仅有 C 时

如果某个节点前端的信号源或电源源阻抗本身已经不小，那么在该节点对地并联一个电容，通常就已经能够形成低通效果。

原因在于，实际电路中总会存在某种前级阻抗，例如：
- 走线阻抗
- 电源内阻
- 驱动器输出阻抗
- 电感/磁珠阻抗
- 电阻本身
- 器件内部等效阻抗

因此，即使原理图中没有显式画出一个 `R`，电路也未必是“纯 C”结构。

#### 2.2 加入 R 时

加入 `R` 的作用，是将这种“前级阻抗”显式化并使其可控。
其优点包括：
- 截止频率更好算
- 滤波效果更稳定
- 还能顺带限流、阻尼
- 可以减小尖峰电流

因此，RC 的价值通常不在于“没有 `R` 就完全不能滤波”，而在于**加入 `R` 后，滤波行为更可控**。

### 3. 为什么电源去耦经常只使用电容

这是因为电源线在很多情况下本来就不是理想零阻抗，尤其在高频下更是如此。
例如，在芯片电源脚附近并联一个 `100nF` 电容时：
- 负载电流突变时，电容就近供电
- 高频噪声通过电容回地
- 不一定非要再串个 R

这就是最经典的去耦方式。
也就是说，**并联电容是最基础、最常见的滤波方式**。
RC 只是其中一种更“刻意设计”的形式。

### 4. 为什么前述位置仍更适合电感或磁珠

仅并联电容当然也能够滤波。一般DC电源希望实现的不仅是“将一部分高频噪声导向地”，还包括：
- 前后级高频隔离
- 阻止后级噪声反灌
- 阻止前级噪声继续传播
- 让两个支路少互相串扰

此时，仅有并联电容往往不够，因为**如果前级路径阻抗过低，高频噪声仍然容易沿线路传播**。

因此，通常还需要在路径上增加一个“阻拦元件”，例如：
- 电感
- 磁珠
- 有时是电阻

而在电源主路径上，为了避免持续压降和发热，通常优先选择**磁珠 / 电感**，而不是**电阻**。

### 5. 一个实用的理解方式

**仅并联 C**
可以理解为在旁路上增加了一个“泄放通道”，使部分高频噪声更容易经由该路径离开主线路。

**再加入 L 或磁珠**
则可以理解为在主路径上增加了一道对高频不友好的“阻拦”，使高频噪声更难继续向前传播，也更倾向于经由旁路电容离开。

因此，**C 的作用偏向泄放**，**L / 磁珠的作用偏向阻拦**。
两者配合通常比单独使用电容具有更强的滤波效果。

### 6. 为什么很多地方“只放电容就够了”

原因在于需求不同。
例如：
- 芯片电源脚边的去耦
- 参考电压旁路
- 模拟节点小信号滤波

这些位置可能只需要实现以下目标：
- 本地储能
- 抑制高频尖峰
- 缩小瞬态电流回路

此时，一个电容往往已经足够，且方案简单有效。
但如果希望进一步实现：
- 支路隔离
- 更强的电源净化
- 抑制模块间串扰

则常会看到如下结构：
- 磁珠 + 电容
- 电感 + 电容
- RC
- π 型滤波

### 7. 一句话总结

**滤波并不一定需要电阻；在很多场合，直接对地并联一个电容就已经是在滤波。只是当希望滤波效果更强、更可控，或者希望前后级之间实现隔离时，通常还会再加入一个串联阻抗元件，而在电源主路径上，该串联元件通常更适合采用电感或磁珠，而不是电阻。**

这一问题实际上已经涉及“去耦、旁路、滤波、隔离”几个概念之间的边界。下一步值得进一步区分的是：**并联一个电容时，它究竟应称为去耦、旁路，还是滤波，以及三者之间的差别是什么。**

## 安全

## ESD

| 器件 | 特点 | 典型应用 |
| --- | --- | --- |
| **ESD 二极管（Rail-to-Rail）** | 极低结电容（< 1 pF），响应 ps 级 | 高速信号线，USB、HDMI、PCIe |
| **TVS 二极管** | 功率大，钳位精确 | 电源线、低速接口 |
| **肖特基二极管阵列** | 双向保护，集成化 | 多路信号保护 |
| **聚合物 ESD 抑制器** | 极低电容，无残余漏电 | RF 射频电路，天线端口 |

| 层级 | 措施 |
| --- | --- |
| **机械屏蔽** | 金属外壳、屏蔽罩，所有开口加导电衬垫 |
| **接地** | 机壳就近接大地，单点或网状接地 |
| **隔离** | 光耦、磁耦隔离器（如 ISO7241），彻底切断传导路径 |
| **电缆** | 屏蔽电缆，屏蔽层两端接地（或单端，视频率而定） |

## 工控/DCS 场景重点

在电厂、变电站等强电磁环境中，单靠 ESD 器件往往不够，通常采用**分层防护**：

```text
现场设备
   │
  防雷端子（第一级，GDT 或 MOV）
   │
  隔离栅 / 安全栅（光电隔离）
   │
  DCS I/O 模块入口（TVS + RC 滤波）
   │
  背板总线
```

> 核心思路：**能量越大的干扰靠越前的器件处理**，最后进入芯片前电压已被安全钳位。
