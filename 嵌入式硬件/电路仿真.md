# 电路仿真

## 电路仿真软件

> - “SPICE 仿真”主要指基于 SPICE 或兼容 SPICE 的电路仿真工具。
> - “数字”一栏里很多其实是“数字/混合信号（mixed-signal）”，不一定是纯数字逻辑工具。
> - “直观电路原理”一栏更偏教学、交互、快速理解，不一定适合严肃器件级验证。

### 1）SPICE 仿真｜开源｜模拟 / 模拟为主

| 软件 | 主要方向 | 简评 | 链接 |
| --- | --- | --- | --- |
| ngspice | 模拟 + 混合信号 | 经典开源 SPICE 内核，很多前端都拿它做后端 | [ngspice](https://ngspice.sourceforge.io/) |
| KiCad + ngspice | 模拟 + 混合信号 | 原理图与仿真集成，适合边画边仿 | [KiCad SPICE](https://www.kicad.org/discover/spice/) |
| Xyce | 模拟为主 | 高性能、支持大规模并行，偏工程/科研 | [Xyce](https://xyce.sandia.gov/) |
| Qucs-S | 模拟前端 | 图形化前端，本身不是求解器，可接 ngspice / Xyce / Qucsator | [Qucs-S](https://ra3xdh.github.io/) |

### 6）选型建议

- 想替代 LTspice 做开源 SPICE：优先看 **KiCad + ngspice**
- 想做大规模或高性能模拟：看 **Xyce**
- 想做单片机 + 外设 + 原理图联动：看 **Proteus** 或 **Wokwi / SimulIDE**
- 想直观学电路原理：先用 **CircuitJS1 / Falstad**
- 想学数字逻辑：用 **Logisim-evolution** 或 **Digital**
