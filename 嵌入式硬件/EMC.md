# EMC问题

## 简述

EMC也就是在电磁环境中：

- 不要过分骚扰别人：也就是发射问题，叫 **EMI（Electromagnetic Interference，电磁干扰）**
- 不要太容易被别人骚扰坏：也就是抗扰度问题，叫 **EMS / Immunity（电磁敏感度 / 抗扰度）**

从干扰源来分类的话：

- 电源口问题
- I/O 口问题
- 空间辐射问题
- 板内耦合问题

## 常规解决方案

### 电源口问题

加共模电感、X/Y 电容、机壳泄放路径、输入滤波地参考。

### I/O 口问题

屏蔽线怎么接、TVS 泄放、共模电流滤波等。

### 空间辐射问题

把高频电流限制在小回路里，让机壳形成屏蔽面，控制缝隙辐射。

### 板内耦合问题

模拟地、数字地、功率地的组织，敏感信号参考面连续，回流是否绕远路等。

## 当前设计

### 电源
- 电源侧已经考虑了输入浪涌、传导干扰和上电冲击问题，前端有滤波与抑制，后端有钳位与储能，属于比较标准的抗干扰电源入口设计。

### 板内
- 系统内部不是单一地和单一电源到底，而是交流侧、现场输出侧、逻辑采样侧做了分域隔离，这能降低强干扰直接进入 MCU 和采样链路。

### I/O
- 4-20mA 输出通道面向现场应用，重点做了输出口保护和地参考稳定，能降低外部线缆、负载异常和瞬态干扰对功能的冲击。

- 电压采集通道在高压侧先限流，再通过隔离器件和两级滤波进入采样侧，设计上同时考虑了人机安全、器件安全和采样稳定性。

## 安全

### ESD

| 器件 | 特点 | 典型应用 |
| --- | --- | --- |
| **ESD 二极管（Rail-to-Rail）** | 极低结电容（< 1 pF），响应 ps 级 | 高速信号线，USB、HDMI、PCIe |
| **TVS 二极管** | 功率大，钳位精确 | 电源线、低速接口 |
| **肖特基二极管阵列** | 双向保护，集成化 | 多路信号保护 |
| **聚合物 ESD 抑制器** | 极低电容，无残余漏电 | RF 射频电路，天线端口 |

| 层级 | 措施 |
| --- | --- |
| **机械屏蔽** | 金属外壳、屏蔽罩，所有开口加导电衬垫 |
| **接地** | 机壳就近接大地，单点或网状接地 |
| **隔离** | 光耦、磁耦隔离器（如 ISO7241），彻底切断传导路径 |
| **电缆** | 屏蔽电缆，屏蔽层两端接地（或单端，视频率而定） |

## 工控/DCS 场景重点

在电厂、变电站等强电磁环境中，单靠 ESD 器件往往不够，通常采用**分层防护**：

```text
现场设备
   │
  防雷端子（第一级，GDT 或 MOV）
   │
  隔离栅 / 安全栅（光电隔离）
   │
  DCS I/O 模块入口（TVS + RC 滤波）
   │
  背板总线
```

> 核心思路：**能量越大的干扰靠越前的器件处理**，最后进入芯片前电压已被安全钳位。

## ESD 测试标准参考

- **IEC 61000-4-2**：ESD 抗扰度，接触放电 ±4 kV，空气放电 ±8 kV（工业级）
- **IEC 61000-4-4**：电快速瞬变脉冲群（EFT），针对感性负载开关
- **JEDEC JS-001**：芯片级 HBM（人体模型）ESD 测试
